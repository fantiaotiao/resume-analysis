## 三、技术栈（零基础友好）
### 后端（零基础入门级）
- 框架：FastAPI（RESTful API，自动生成接口文档，易上手）
- PDF解析：pdfplumber（开箱即用，无需复杂配置）
- AI能力：通义千问API（调用式使用，无需训练模型）
- 缓存：Redis（本地单机部署，简单验证）
- 部署适配：代码无硬编码，可直接迁移至阿里云FC

### 前端（熟练栈）
- 框架：Vue + Element Plus
- 网络请求：axios
- 核心交互：PDF上传校验、岗位需求填写、分析结果可视化
